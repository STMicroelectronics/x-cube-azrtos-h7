The STM32H735G-DK Discovery kit is a complete demonstration and development platform for Arm® Cortex®-M7 core-based STM32H735IGK6U microcontroller, with 1 Mbyte of Flash memory and 564 Kbytes of SRAM.


The STM32H735G-DK Discovery kit is used as a reference design for user application development before porting to the final product, thus simplifying the application development. The full range of hardware features available on the board helps users to enhance their application development by an evaluation of all the peripherals (such as USB OTG FS, Ethernet, microSDT card, USART, CAN FD, SAI audio DAC stereo with audio jack input and output, MEMS digital

 
microphone, HyperRAM, Octo-SPI Flash memory, RGB interface LCD with capacitive touch panel, and others). ARDUINO® Uno V3, PmodT and STMod+ connectors provide easy connection to extension shields or daughterboards for specific applications. STLINK-V3E is integrated into the board, as the embedded in-circuit debugger and programmer for the STM32 MCU and USB Virtual COM port bridge. The STM32H735G-DK board comes with the STM32CubeH7 MCU Package, which provides an STM32 comprehensive software HAL library as well as various software examples.

 About this application

 

This application is part of the Azure RTOS STM32Cube expansion package for STM32H7 series, it presents HTTP webserver developed based on the USBX CDC ECM class, Azure RTOS NetX Duo TCP/IP network stack and FileX file system stack, it allows the loading of a static home web page from sd card and led control web page.

Application main features:

  • Azure RTOS USBX
      • Class: Device CDC ECM.
  • Azure RTOS NetX Duo
      • Protocols: IP, ICMP, ARP, UDP, TCP.
      DHCP client (dynamic IP address allocation).
      Addons/Web HTTP.
  • Azure RTOS FileX API (to fetch the html page from sd card).


For more information you can refer to Azure RTOS documentation  Azure RTOS USBX & Azure RTOS NetX Duo & Azure RTOS FileX
